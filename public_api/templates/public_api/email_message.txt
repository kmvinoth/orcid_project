    <h5>Was ist ORCID?</h5>
    Die Open Research and Contributor iD (ORCID) ist eine Non-Profit-Organisation, die eindeutige Identifizierungsnummern (ORCID IDs) für Forschende vergibt, die es ermöglichen, Wissenschaftler*innen mit ihrer Forschung zu verknüpfen. Die iD ist verlagsunabhängig und kann dauerhaft und institutionsunabhängig von Forschenden für ihren wissenschaftlichen Output verwendet werden. Die Registrierung erfolgt über https://orcid.org und ist grundsätzlich für alle offen. Bei einem Wechsel der Universität behalten Forschende Ihre ORCID ID und können sie weiter nutzen.

    <h5>Warum ORCID?</h5>
    Mit Hilfe einer ORCID ID können Forschende eindeutig Identifiziert werden und eine Verbesserung der Sichtbarkeit der Forschung erreicht werden. Publikationslisten lassen sich einfach erstellen und verwalten. Durch die Ernennung von sogenannten trusted parties‘ lassen sich Publikationsmetadaten automatisch in die ORCID Profile der Forschenden übernehmen, ggf. auch in das institutionseigene Publikationssystem (Repository, FIS, oder Hochschulbibliografie).

    <h5>What is ORCID?</h5>
    Open Research and Contributors iD (ORCID) is a non-profit organization that provides researchers with unique identification numbers (ORCID IDs) that allow scientists to link to their research. ID is independent of publishers and can be used by researchers for their scientific output on a permanent and institution-independent basis. Registration is via https://orcid.org and is open to all. If the university changes, researchers retain their ORCID ID and can continue to use it.

    <h5>Why use ORCID?</h5>
    With the help of an ORCID ID, researchers can be clearly identified and an improvement in the visibility of research can be achieved. Publication lists are easy to create and manage. Through the appointment of so-called trusted parties, publication metadata can be automatically transferred to the ORCID profiles of the researchers, possibly also into the institution's own publication system (repository, FIS, or university bibliography).
